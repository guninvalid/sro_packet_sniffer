### TL;DR: How To Use

You will need to import `scapy`. That should be the only dependence you need.

Run the sniffer using `python sniffer.py` or `python3 sniffer.py`. Make any behavioral configurations using `config.py`.

# Soul's Remnant Sniffer Client

This is a sniffer client meant to decrypt and understand packets sent to and from the Soul's Remnant client. There are several missing features I'd like to add, and I'll try to keep a todo list here if I come back to the project.
